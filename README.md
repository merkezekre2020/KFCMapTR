# 🍗 KFCMapTR - Türkiye KFC Şubeleri Haritası

Türkiye'deki tüm KFC şubelerini interaktif harita üzerinde gösteren web uygulaması.

![KFC Türkiye](https://img.shields.io/badge/KFC-Türkiye-red)
![Şube Sayısı](https://img.shields.io/badge/Şube-25-brightgreen)

## ✨ Özellikler

- 🗺️ Tüm KFC şubelerini harita üzerinde gösterme
- 🔍 Şube adı, şehir ve ilçe ile arama
- 📦 Paket servis ve gel-al seçeneklerini görme
- 📱 Mobil uyumlu tasarım

## 🚀 Kullanım

1. Repo'yu klonlayın:
```bash
git clone https://github.com/merkezekre2020/KFCMapTR.git
```

2. `index.html` dosyasını tarayıcıda açın.

## 📊 Veri Kaynağı

Şube verileri [kfcturkiye.com](https://www.kfcturkiye.com/restoranlar) adresinden çekilmiştir.
